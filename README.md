# DVC

### Steps to follow:
  1. Create a folder mlops/DVCDemo
     ```
     mkdir mlops/DVCDemo
     ```
  2. Create a new environment
     ```
     conda create -p mlops python==3.9 -y
     ```
  3. Create a file .gitignore
  4. Create a file requirements.txt
     ```
     nano requirements.txt
     ```
     Put contents as below:
